# Friend Connect Tracker

## Description
Friend Connect Tracker is a Next.js app for managing personal relationships.  
It helps you view friend details, do quick check-ins (Call, Text, Video), and track those interactions in a timeline view.

## Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS
- React Icons
- React Toastify

## Features
- Browse friend cards from a JSON data source
- Open a dedicated friend details page using dynamic routing
- Perform Quick Check-In actions (Call, Text, Video)
- Show instant toast notifications after each check-in action
- View timeline updates dynamically with filter support (All/Call/Text/Video/Meetup)
- Display a clear empty state (`No Check in`) when there are no events
